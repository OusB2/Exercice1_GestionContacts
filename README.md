# Exercice 1 : Gestionnaire de Contacts (ContactsManager)

# Description
Ce projet est une application Java simple réalisée dans le cadre du cours sur la Programmation Orientée Objet (POO). 
Le projet est composé de trois classes principales :
1. **Contact.java** : Définit les attributs d'un contact (nom et numéro de téléphone).
2. **ContactsManager.java** : Contient la logique pour stocker les contacts dans un tableau et effectuer des recherches.
3. **Main.java** : Point d'entrée du programme où l'on crée les objets et testons les fonctionnalités.

## Fonctionnalités
- **Création de contacts** : Instanciation de plusieurs objets `Contact`.
- **Gestion de liste** : Ajout des contacts dans un tableau via un objet `ContactsManager`.
- **Recherche** : Recherche d'un contact par son nom et affichage de son numéro de téléphone dans la console.

## Instructions d'installation
1. Clonez ce dépôt ou téléchargez les fichiers.
2. Importez le projet dans votre IDE (ex: IntelliJ IDEA).
3. Exécutez la classe `Main.java` pour voir le résultat.
